# 第6章 禁止事項

以下の行為は明示的に禁止される。違反した場合は就業規則に基づく処分の対象となる場合がある。

> **根拠**: IPA p.38-42

### 6.1 入力に関する禁止事項

| 禁止事項 | 具体例 |
|---------|-------|
| 特定個人情報の入力 | マイナンバー、基礎年金番号 |
| 顧客許可のない顧客データの入力 | 顧客のソースコード、顧客DB情報 |
| 機密情報の無断入力 | 未発表製品情報、M&A情報、経営戦略 |
| 学習許可状態での機密入力 | オプトアウト未設定ツールへの社外秘以上の入力 |
| 著作権侵害コンテンツの入力 | 他者の著作物を無断でプロンプトに含める |

### 6.2 出力に関する禁止事項

| 禁止事項 | 具体例 |
|---------|-------|
| 生成物の無検証での外部公開 | ハルシネーション確認なしでの報告書提出 |
| 生成物を自己の著作物と偽ること | AI生成であることを隠して納品 |
| 有害コンテンツの生成・利用 | 差別的表現、暴力的表現、違法行為の助長 |
| 第三者権利侵害コンテンツの利用 | 類似性確認なしでの商標・意匠の利用 |

### 6.3 学習・共有に関する禁止事項

| 禁止事項 | 具体例 |
|---------|-------|
| 無許可でのファインチューニング | 顧客データを使った無断学習 |
| プロンプト・テンプレートの無断外部共有 | 社内ノウハウを含むプロンプトのSNS投稿 |
| 社内限定ナレッジの外部漏洩 | RAG用社内文書の外部持ち出し |

### 6.4 アカウント・端末に関する禁止事項

| 禁止事項 | 具体例 |
|---------|-------|
| アカウントの貸与・共有 | 自分のIDを他者に使わせる |
| 未承認ツールの利用 | 許可リストにないAIサービスの業務利用 |
| 私用端末での業務利用 | BYODポリシーに反する利用 |

### 6.5 セキュリティに関する禁止事項

| 禁止事項 | 具体例 |
|---------|-------|
| プロンプトインジェクション攻撃の実行・加担 | システムプロンプト抽出の試行 |
| セキュリティ機能の無効化 | ガードレールの迂回 |
| 脆弱性の悪用 | 既知の脆弱性を利用した攻撃 |
