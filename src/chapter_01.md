# 第1章 目的・適用範囲・用語定義

### 1.1 目的

本ガイドラインは、当社における生成AI（Generative AI）の導入・利用・開発・提供において、安全性の確保、法令遵守、リスク管理を徹底し、AIの便益を最大化しつつ社会的責任を果たすことを目的とする。

> **根拠**: 総務省AI事業者ガイドライン p.2-3「AIの安全安心な活用が促進されるよう、我が国におけるAIガバナンスの統一的な指針を示す」

### 1.2 適用範囲

本ガイドラインは、以下の者に適用される。

| 対象者 | 適用内容 |
|-------|---------|
| 全役職員（正社員・契約社員） | 全条項 |
| 派遣社員 | 業務上利用する場合、全条項準用 |
| 業務委託先 | 契約に本ガイドライン遵守条項を含める |
| インターン・アルバイト | 区分1の利用に限り適用 |

### 1.3 用語定義

| 用語 | 定義 | 根拠 |
|-----|------|------|
| **生成AI（Generative AI）** | 文章、画像、プログラム等を生成できるAIモデルにもとづくAIの総称 | 総務省 p.10 |
| **LLM（大規模言語モデル）** | 大量のテキストデータを学習した大規模な言語モデル。GPT、Claude、Gemini等 | IPA p.14 |
| **AIエージェント** | 自律的にタスクを実行し、外部ツールと連携して目標を達成するAIシステム | 推測（一般的定義） |
| **RAG（検索拡張生成）** | 外部データベースから関連情報を検索し、回答生成に活用する技術 | IPA p.15-16 |
| **ファインチューニング** | 学習済みモデルに対して追加データで再学習を行い、特定分野に特化させる技術 | IPA p.15 |
| **プロンプト** | 生成AIへの入力となる指示文・質問文 | IPA p.16-17 |
| **ハルシネーション** | 生成AIが事実に基づかない情報を生成する現象 | IPA p.41 |
| **AIガバナンス** | AIのリスクを受容可能な水準で管理しつつ、便益を最大化するための技術的・組織的・社会的システムの設計及び運用 | 総務省 p.10 |
| **AI開発者** | AIシステムを開発する事業者 | 総務省 p.5 |
| **AI提供者** | AIシステムをサービスとして提供する事業者 | 総務省 p.5 |
| **AI利用者** | 事業活動においてAIシステム・サービスを利用する事業者 | 総務省 p.5 |
