## 本書について

本ガイドラインは、当社における生成AI（Generative AI）の導入・利用・開発・提供に関する包括的な指針を定めた文書です。

### 構成

| 章 | タイトル | 概要 |
|---|---------|------|
| 第1章 | 目的・適用範囲・用語定義 | ガイドラインの目的、適用対象、主要用語の定義 |
| 第2章 | 基本原則 | 生成AI活用における8つの基本原則 |
| 第3章 | ガバナンス体制 | 責任者、審査会議、意思決定プロセス |
| 第4章 | 利用区分 | 3つの利用区分とその適用ルール |
| 第5章 | データ分類と取り扱い | データ分類基準と入力可否マトリクス |
| 第6章 | 禁止事項 | 明示的に禁止される行為 |
| 第7章 | リスク評価と事前審査 | リスクアセスメント手法と審査基準 |
| 第8章 | セキュリティ要件 | 認証・権限・ログ・データ保護 |
| 第9章 | 品質・安全性要件 | ハルシネーション対策、バイアス対策 |
| 第10章 | 法務・知財・契約 | 著作権、個人情報保護、責任分界 |
| 第11章 | 運用ルール | 利用申請フロー、台帳管理 |
| 第12章 | 事故・インシデント対応 | インシデント分類と報告フロー |
| 第13章 | 教育・周知 | 教育対象者と教育内容 |
| 第14章 | 監査・見直し | 監査KPIと定期点検 |
| 第15章 | 付録 | FAQ、ツール設定ガイダンス、サンプルプロンプト集 |
