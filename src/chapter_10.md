# 第10章 法務・知財・契約

## 10.1 著作権

### 10.1.1 入力時の注意
- 他者の著作物を無断でプロンプトに含めない
- 引用する場合は著作権法の要件を満たす

### 10.1.2 出力物の取り扱い
- 生成AIの出力は、そのままでは著作物に該当しない場合がある
- 人間の創作的関与がある場合は著作物となりうる
- 既存著作物との類似性を確認する

> **根拠**: IPA p.42

## 10.2 個人情報保護

- 個人情報の入力は原則禁止
- 匿名化処理を行った上で利用
- プライバシーポリシーへのAI利用の明記

> **根拠**: 総務省 p.17-18

## 10.3 秘密保持

- NDA締結済み案件での情報入力可否を確認
- オプトアウト設定の確認
- 契約違反とならないことの確認

## 10.4 責任分界

### 10.4.1 受託開発における責任分界

```
【当社の責任範囲】
- 契約に定めた仕様どおりのAI機能開発
- 開発段階でのセキュリティ対策
- 納品物のドキュメント整備
- 瑕疵担保責任（契約条件による）

【顧客の責任範囲】
- 運用環境の構築・維持
- 運用段階でのデータ管理
- エンドユーザーへの利用規約整備
- AI出力の最終判断・利用
```

### 10.4.2 SESにおける責任分界

```
【当社の責任範囲】
- 技術者の労働者派遣法遵守
- 技術者のスキル・コンプライアンス教育
- 本ガイドラインの技術者への周知

【顧客の責任範囲】
- 指揮命令下での業務指示
- 顧客環境でのセキュリティ管理
- 顧客ポリシーの提示
```

### 10.4.3 コンサルティングにおける責任分界

```
【当社の責任範囲】
- 専門的見地からの助言・提案
- 情報の正確性についての合理的な確認
- 秘密情報の適切な管理

【顧客の責任範囲】
- 助言に基づく最終的な意思決定
- 実装・運用の実行
- 実装結果の検証
```

## 10.5 免責条項テンプレート

```
【AI機能に関する免責事項】
1. AI機能の出力は参考情報であり、最終的な判断は利用者が行うものとします。
2. AI機能の出力の正確性、完全性、適時性について、当社は保証しません。
3. AI機能の利用により生じた損害について、当社の故意または重過失による場合を除き、
   当社は責任を負いません。
4. AI提供者（モデル提供者）の利用規約変更、サービス停止等により
   AI機能が利用できなくなった場合、当社は責任を負いません。
```

> **根拠**: 推測（一般的実務）、法務確認要
